APP Store安装正版的旧版APP
发表于 2020-04-27| 更新于 2020-05-01|系统IOS
阅读量:|评论数:

APP Store安装正版的旧版APP（以Thor 1.3.4版本为例）

搜索规则关键字 f4thor

此方法可能不适用于 部分软件的外区~~ （不适用于IOS旧版本下载程序）

本文参考： Thor 1.3.4 正版下载

零、流程

  1. 用老版本的iTunes （iTunes 12.6.5.3 ）下载软件（以Thor为例）
  2. 找到下载好的APP，用Plist 编辑器 修改他的版本信息为1.3.4的版本号，并保存
  3. 删除iTunes–资料库中的Thor图标
  4. 再把修改好版本好的APP拖到资料库中，刷新下，会显示更新。
  5. 用旧版本下载程序，锁定版本号，在更新中更新APP（我猜是因为要伪装它是下载的APP，而不是外来的APP，这样一个动作是修复ipa包里的完整性？）
  6. 要第三方软件安装APP即可。

一、下载工具，并且下载软件

  1. 下载老版本iTunes， 12.6.5.3 下载地址：

    • 64位：http://t.cn/EPpDk07
    • 32位：http://t.cn/EPpDkpk

  2. 安装iTunes，登录你的账号—授权这台电脑–并且搜索软件，下载（以Thor 为例）

  3. 下载后的APP会在资料库中显示

  4. 下载 Plist 编辑器 Plist Editor for Windows

    • 官网 ，临时用

  5. IOS旧版本下载程序-IOS免越狱安装历史版本APP

    • 论坛地址

    • 下载地址

    • 找旧版APP下载地址的 原理

二、修改APP的版本号为旧版本号

  1. 打开Plist 编辑器

  2. 资料库 - 右键，选择Thor - 在Windows资料库中显示

  3. 用RAR打开Thor.ipa文件

    • 技巧：先右键压缩下，然后打开压缩包，点三个点，返回上级目录，双击Thor.ipa就能打开

  4. 找到iTunesMetadata.plist文件，拖到 Plist 编辑器中

    • 如果拖不动，就把plist文件从rar中拖出来，再用编辑器打开
    • 等下编辑好后，再拖会RAR中

  5. Ctrl+F 搜索 softwareVersionExternalIdentifier ，下一行的数字就是本软件的版本号

    • 技巧：怎么查版本号
      • 打开IOS旧版本下载程序
      • 开启
      • 在iTunes中复制下软件 APP，IOS旧版本下载程序就能列出该软件的版本号

  6. 把下一行的版本号，改成你要的版本的版本号。比如 Thor1.3.4的版本号是 827080419

  7. 保存 （请确认文件修改成功!）

三、修复IPA文件的完整性

下面的操作，是不是因为 完整性，完全个人猜测

  1. iTunes – 资料库 —- 右键Thor — 从资源库中删除 — 删除应用 — 保留文件
  2. 把刚刚修改号的Thor 的IPA文件拖到 资料库 中
  3. F5刷新下，等一下，Thor图标上会有 更新 字样
  4. 打开 旧版本下载程序 — 开启
  5. iTunes – Thor 商店主页中 — 更新的右边下三角—-复制链接
  6. 旧版本程序– 选择你要的版本号 比如：1.3.4 ，然后锁定
  7. iTunes — 更新 —- 点下Thor图标 —- 更新
    • 一定要看到它下载下来！
  8. 最后用第三方软件安装即可。 比如iTools

旧版本程序 状态是开启时，可能无法连接苹果ID服务器，及无法使用复制粘贴，暂时先关闭即可
