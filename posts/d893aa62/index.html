CentOS7.8安装及更新换国内源

安装CentOS

步骤

更改yum源为国内源

步骤：

  1. 备份原有源
  2. 下载 当前系统版本的 国内源（阿里、163、清华等等）
  3. 清空yum缓存 （yum clean all）
  4. 生成缓存cache （yum makecache）
  5. 更新yum （ yum update -y）

使用脚本自动更换

感谢 秋叶依剑的脚本

  1. 获取脚本 centos.sh 文件，有以下三个方法

    • 方法一：wget下载sh文件 wget -O centos.sh "https://qiuyeyijian.com/download/bash/centos.sh"
    • 方法二：Git下载作者的库 git clone https://gitee.com/qiuyeyijian/bash.git，记得下载后，要cd到sh的文件夹内
      • Gitee
      • GitHub
    • 方法三：自己建立 centos.sh 文件，输入上面的脚本代码保存，然后ftp传输到CentOS中即可

  2. 然后在含centos.sh 文件的文件夹下 输入：chmod +x centos.sh && ./centos.sh

手动更换源

  1. 获取当前系统版本 ：cat /etc/centos-release
  2. 进入yum源配置目录 ：cd /etc/yum.repos.d/
  3. 备份yum源 ：mv CentOS-Base.repo CentOS-Base.repo.bak
  4. 下载国内源 ： wget -O /etc/yum.repos.d/CentOS-Base.repo 对应系统版本源地址
    • 阿里
      • CentOS 5http://mirrors.aliyun.com/repo/Centos-5.repo
      • CentOS 6http://mirrors.aliyun.com/repo/Centos-6.repo
      • CentOS 7http://mirrors.aliyun.com/repo/Centos-7.repo
    • 163
      • CentOS 5 http://mirrors.163.com/.help/CentOS5-Base-163.repo
      • CentOS 6 http://mirrors.163.com/.help/CentOS6-Base-163.repo
      • CentOS 7 http://mirrors.163.com/.help/CentOS7-Base-163.repo
  5. 清除缓存：yum clean all
  6. 生成缓存：yum makecache
  7. 更新软件：yum update -y
