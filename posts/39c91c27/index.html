玩转京东自动化脚本

一、获取京东Cookies

Cookies可能需要一个月左右一次抓包一次

手机方法

此方法适用于 苹果捷径方法

  1. 找一个抓包软件，比如 Thor
  2. 开启抓包软件Thor的 全局抓包 模式，开始抓包
  3. 手机浏览器登录 bean.m.jd.com，进行登陆
  4. 登陆成功后，返回抓包软件，进入抓包记录
  5. 找到 Client.action - 请求- 点击 请求头 - 长按 Cookies，复制

电脑Chrome开发者工具方法

  1. 电脑上打开 Chrome 浏览器，按 F12 打开开发者工具，并点击工具栏的左上角选择手机模式
  2. 打开网页 bean.m.jd.com，进行登陆
  3. 登陆成功后，选择右侧工具栏中的 console
  4. 然后在下放输入 copy(document.cookie) 它会拷贝当前的 cookie，
  5. 然后在一个文本编辑器中或其他可编辑的窗口中复制出内容，并保存好，下方要用。
  6. 点击 Network ，清空 Network 下的请求
  7. 在请求搜索框搜索 QueryUserInfoM
  8. 打开该请求，找到 Request Headers 节点下面，复制完成的 cookie

电脑Chrome的EditThisCookie插件方法

此方法适用于部分脚本

插件获取京东cookie教程

  1. 安装EditThisCookie插件 ：打开此网站 进行安装
  2. 打开京东，并登陆（需要保持登陆状态）
  3. 点击插件 EditThisCookie，
    1. 搜索key,复制pt_key的value值
    2. 搜索pin，复制pt_pin的value值
  4. 组合pt_key和pt_pin
    1. pt_key=复制插件搜索出来的key值;pt_pin=复制插件搜索出来的pin值; ,后面的英文引号;是必须要的
