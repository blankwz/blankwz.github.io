Vscode之必装插件

一、如何配置插件

部分插件安装完成后，需要配置插件

  1. 进入命令面板：查看 - 命令面板或者 Ctrl + shift + P
  2. 搜索 ：settings.json
  3. 根据你的插件方法，写入命令

二、代码优化

0. codelf 变量命令神器

安装：搜索 codelf

使用：只需要选中变量名，然后右键选择CodeIf就可以跳转到网页，显示候选命名

1. Yapf

Yapf是谷歌开源的一个用于格式化Python代码的工具，可以一键美化代码。支持两种规范：PEP8和Google Style

安装：pip install yapf

开启：在settings.json文件中输入"python.formatting.provider": "yapf"

试试： 按下快捷键 Alt+Shift+F 即可自动格式化代码

2. vscode-fileheader

使用该插件可以给自己的文件添加文件头，这样更方便维护自己的代码的权益

  • 用户在settings.json中配置用户名和最后修改用户名即可使用
    • “fileheader.Author”:"Gxkon"
    • “fileheader.LastModifiedBy”:"Gxkon"
  • 使用：Ctrl+Alt+I

3. indent-rainbow

这个插件是用五颜六色的色块来对代码的缩进做区分

  • 使得对齐更加具有可读性

4. Bracket Pair Colorizer 2:

  • 可以方便查看每一对{}、()与谁配对

5. Prettier - Code formatter

  • 保存即可格式化代码的神器
  • 配置：设置 - 搜索 format on save - 勾选

6. Trailing Spaces

高亮那些冗余的空格，可以快速删掉

7. black for autoFormatting

只需要在配置文件里面一键设置每次保存代码的时候去formatting，效率那是大大的快！

安装：pip install black

设置：在settings.json文件中输入

Github：地址

8. Linting

linting和formatting是相互补充的一个东西，linting会提示你的代码哪里有问题

配置：看这

9. 代码检查工具 ESLint

ESLint 是一个语法规则和代码风格的检查工具，可以用来保证写出语法正确、风格统一的代码

10. 代码拼写检查 Code Spell Checker

此插件安装后就不用管就好了，在你代码中有单词拼写错误时，你就会发现它的好处

11. Better Align

一款能够实现代码规范的工具，它主要用于代码的上下对齐

它能够用冒号（：）、赋值（=，+=，-=，*=，/=）和箭头（=>）对齐代码

使用方法：选中代码，Ctrl+Shift+p输入“Align”确认即可。

12. Path Intellisense

Path Intellisense是一款自动完成文件名、文件路径的 VS Code 插件。

Path Intellisense 就可以让你像补全代码一样去不全文件名，这样就会大大提升开发过程中的效率。

三、强化注释

1. docstring

安装：插件商店，搜索docstring

试试：你只需要在函数名后面输入三个双引号然后回车，即可生成docstring注释

2. Better Comments

主要是针对行注释，对不同类型的注释会附加不同的颜色，更加方便区分！

  • 主要使用！、？、TODO区分

3. Colorful Comments

Colorful Comments是一款能够让你轻松创建更加易于阅读、对用户更加友好注释的一款VS Code插件。

使用Colorful Comments时，我们可以用不同的符号来区分注释的类型，然后VS Code就可以用不同的颜色来区分它们，这样大大提升了注释的辨识度。

四、更强大的自动补全

1. Kite

专门用AI来做代码补全，写起代码来会快一些

安装

  1. 插件商店，搜索Kite
  2. 官网下载 Kite程序，安装它

五、大型辅助

1. Drawio

http://draw.io 是一个支持在线绘图的网站工具，因其界面简洁直观，功能丰富强大而受到不少用户喜爱。

2. vscode-leetcode

让你可以在 VSCode 上刷 LeetCode

六、Python插件

1. Pylance

  1. 显示文档字符串和参数
  2. 显示多余的库，变量和函数
  3. 报告代码中的错误

2. Python Preview

展示Python中内置变量的值，方便有时候调试展示值。

3. flake8

flake8 是 Python 官方发布的一款静态代码检查工具

安装：pip install flake8

配置：文件->首选项->用户设置，setting.json中使用

七、前端

1. Prettier

一键美化你的JavaScript/TypeScript/CSS代码

2. npm Intellisense

可以在导入语句自动补全npm模块名称

3. Color Highlight

直观展示你定义的颜色

4. CSS Peek

可以在HTML中通过CSS id或则class来定位到其定义

5. Quokka.js

实时执行JavaScript代码(做快速的demo很有用)。

6. Vetur

VS Code下面的Vue工具！有Pine Wu开发，已经累计400多万下载量！

7.arrr

arrr是一款在前端开发过程中非常高效的一款插件，它能够快速把HTML代码提取成一个组件。

八、Debugger

1. Debugger for Chrome

此工具简直就是前端开发必备，将大大地改变你的开发与调试模式

配置挺复杂的，可以网上找找教程

配置方法：

  1. Ctrl + Shift + P,

  2. 打开launch.json

  3. 点击右下角 - 添加配置

  4. 选择Chrome:launch

  5. 配置启动页面路径

  6. 选择自己配置调试名称，Launch index.html (disable sourcemaps)

  7. 调试

美化

1. 中文包 Chinese(Simplified)Language

VScode默认安装后界面为英文，对于英文不是很好的同学，建议安装VScode后，应先安装此插件。

2.文件图标 vscode-icons

安装：插件商店，搜索 vscode-icons

3. 主题Material Theme

4. *One Dark Pro *暗色主题

一键将 VSCode 编辑器的颜色调整成暗色系，编码起来更加舒适。

5. 字体（不属于插件） Fira Code

  1. 下载字体

  2. 解压，打开ttf文件夹，全选 - 右键 - 安装
