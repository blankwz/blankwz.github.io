Hexo教程一之安装Hexo并链接到Github

Hexo是一款基于Node.js的静态博客框架，依赖少易于安装使用，可以方便的生成静态网页托管在GitHub和Heroku上，是搭建博客的首选框架。这里我们选用的是GitHub

技巧：如果下载主题太慢，就找国内的镜像，或者导入到码云，通过码云来下载~！

一、官方资料

  • 官方中文文档
  • 官方主题库
  • 官方插件库

二、本地安装Hexo

以下所有命令，到在终端或者CMD、PowerShell中执行

  1. 安装Node.js和Git
    • Node.js地址
    • Git地址
  2. 安装Hexo
    • npm install -g hexo-cli
  3. 本地随便找个地方，新建一个站点文件夹
  4. 初始化并安装所需
  5. 测试下
  6. 完成后，打开浏览器输入地址 localhost:4000

三、发布网站到Github平台 （链接Github）

上面只是在本地预览，接下来要做的就是就是推送网站，也就是发布网站

1. Github建立仓库

仓库名格式必须是 用户名.http://github.io
这个用户名使用你的GitHub帐号名称代替，这是固定写法

2.安装git并链接到Github

3.链接到Github

  1. 打开站点的（根目录下）配置文件_config.yml（翻到最后）

  2. 安装Git部署插件，输入命令
    npm install hexo-deployer-git --save

  3. 测试下，分别输入三条命令

  4. 浏览器地址栏输入你的放置个人网站的仓库路径:XXX.github.io

  5. 完成

三、更换域名，绑定域名

  1. 申请一个域名
  2. 打开你的域名服务商平台，进入解析
  3. 加入几条条解析
  4. 进入Github，进入之前创建的仓库，settings---Options---Github Pages--Custom domain，输入你的域名。保存
  5. 本地博客文件夹中，/source目录下，创建一个命名为 CNAME 的记事本文件，输入你的域名， 保存为所有文件，而不是txt文件
  6. 访问域名看看，域名平台的CDN是否解析成功！新添加的CDN需要几十分钟解析！

四、更换主题，选择一个主题（可选）

快速了解的最好方法，就是换一个主题，按主题文档走一遍，基本是就了解了大概

1.更换主题

  1. 官网中选择一个主题，复制他的Github地址
  2. 在博客根目录下，输入
    git clone https://github.com/ABC/XXX themes/XXX
  3. 打开站点的_config.yml配置文件，修改主题名，
    • 比如next主题：theme:next
  4. hexo g、hexo s，查看效果一下吧

六、选择一个图床

七、发布文章

  1. 文章存储路径：
    source/_post/博客名字.md
  2. 快速生成文章方式：
    通过命令（ hexo n "博客名字" ） 就会快速生成文章，仅仅只有一个标题
  3. 写文章方式
    1）直接通过hexo n方式新建文章，然后通过编辑器手动书写文章
    2）直接在_post下新建文章，编辑即可
  4. 本地预览网页：
    命令： hexo s --debug
    浏览器打开：localhost:4000
  5. 生成HTML并且发布到网络上
    命令：hexo g、hexo d
