jar包做成Windows服务启动

需求

把springboot项目打成jar包放到windows服务中启动，做一个jar包部署到现场服务器，做一个bat批处理文件，一键启动，如果服务器重启还需要手动去启动bat文件，而且启动后小黑框一直在桌面上，如果不小心关啦，还需要重新启动。所以要把jar包做成windows服务，服务器重启后就会自动启动这个服务，小黑框也会小时，把服务放到后台运行

方法一：Winsw

Winsw是一款可将可执行程序安装成Windows Service的开源小工具

  • 注意： 要使用winsw，电脑上必须已经安装.NET framework

Winsw Github：

  • 说明
  • 下载
    • 下载 WinSW.NET4.exe

步骤

  1. 将WinSW.NET4.exe文件复制到java程序所在文件夹中

  2. 将java程序重命名，去掉名称中的“.”。例如test-1.0.jar —-> test.jar

  3. 将WinSW.exe重命名为test.exe(和jar同名)

  4. 编写xml (xml、jar包、exe的名字全一样)

  5. 如果没有配置环境变量，直接将(exe、jar、xml)三个文件扔到java的bin目录下运行。

    • 去掉标签<env name="JAVA_HOME" value="%JAVA_HOME%"/>

  6. 命令行定位到当前目录，以管理员身份执行cmd，执行：test.exe install，将应用注册为服务

  7. 启动服务，三种方法：

    1. 去windows服务列表中启动程序

    2. 启动服务停止服务也可以在任务管理器里面手动执行

    3. 以管理员身份执行cmd ,net start 服务名

  8. （如果需要更新程序，只需要先将服务停止，再将新文件重命名为test.jar，最后启动服务就行了）

  9. 程序日志在程序本身目录

其他命令

除了install命令外，还有其它命令：

  • uninstall：删除服务
  • start：启动服务
  • stop：停止服务
  • restart：重启服务
  • status：输出当前服务的状态

方法二：JavaService

1、下载JavaService.exe

JavaService是一款能把Java应用程序做成NT服务的实用工具，一般用于启动应用服务器。
JavaService下载地址：
http://download.forge.ow2.org/javaservice/

  • 64位，下载 JavaService_2_0_10-amd64
  • 32位，下载 JavaService_2_0_10-amd64-src

下载下来后是个压缩包，我们只用里面的javaservice.exe文件就可以

2、配置环境变量

  1. 基本的jdk环境变量就不说啦。

  2. 配置JAR_HOME，创建一个文件夹，例如：JAR_HOME/bin，作为jar包路径，把jar包放入bin目录下。（其实就是配一下路径方便我们再bat文件中引用，也可以不配这个）

  3. 把Javaservice.exe文件/JAR_HOME/bin目录下，和jar包一个路径下。

  4. 在bin目录下新建一个logs文件夹，用于存放日志。

3、命令

  1. 打开cmd命令窗口（管理员身份）,进入bin的根目录，命令：cd %JAR_HOME%（jar包放到哪个盘下，先进入哪个盘下再执行）

  2. 执行JavaService服务命令，注意，下面的命令执行时不要换行，每一条之间有个空格就行

  3. 可以自己更改服务名和 环境变量名

4、开启，关闭，删除服务的命令

5、做成bat批处理

下面是把这些创建服务，开启和关闭服务的命令写成bat批处理文件，操作更方便，

  • 但是要用管理员身份运行，
  • 把这些文件和jar包，JavaService.exe文件放到同一个目录下面

4. 将start.exe注册为系统服务
