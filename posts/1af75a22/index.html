三、运行主程序

因为aria2c.exe是命令程序，运行和终止等等都很不直观，并且很麻烦，还好Windows有bat，就用Bat来控制他吧
麻烦是因为aria2c.exe需要挂着配置文件运行~比如：aria2c.exe --conf-path=aria2.conf 每次输入就很麻烦

不想自己编辑的，可以下载网友们的 懒人包
非常感谢这位 网友
下载懒人包的朋友，可以跳过这步

下面所有文件，都放入aria2c.exe同一个目录下

  1. 运行程序 Start.bat

  2. 停止 Stop.bat

  3. 状态查看 Status.bat

  4. 设为开机启动 Boot.bat

  5. 重启并隐藏窗口启动 Restart.bat

  6. 隐藏窗口启动 Start.vbs

全部添加(下载)完后，点击 Start.bat 运行，程序即启动。
可以把Start.vbs 放到Windows的启动文件夹中，以后就可以不需要打开aria2c.exe了（别忘了加入aria2c.exe的绝对路径,或者文件夹加入环境变量）

四、运行Web管理程序

  1. 下载解压后的Web管理程序，到根目录下，打开index.html

  2. 设置JSON-RPC Path

    • 如果是AriaNG
      • AriaNg会自动连接，不用设置，连接成功后，Aria2 Status显示Connected （前提是运行了aria2c.exe或者Start.bat）
    • Yaaw
      • 如果显示 Unknown protocol, 则点击右上角的设置。设置Path为 http://localhost:6800/jsonrpc

  3. 现在可以直接下载了

五、JSON-RPC Path

  • JSON-RPC Path 默认为: http://localhost:6800/jsonrpc
  • 如果提示 Aria2 RPC 服务器错误 按照以下方法修改
    • host: 指运行 Aria2 所在机器的 IP 或者名字
    • port: 使用 –rpc-listen-port 选项设置的端口, 未设置则是 6800
    • 普通情况设置为: http://host:port/jsonrpc
    • 使用 –rpc-secret=xxxxxx 选项设置为: http://token:xxxxxx@host:port/jsonrpc
    • 使用 –rpc-user=user –rpc-passwd=pwd 选项设置为: http://user:pwd@host:port/jsonrpc
    • 以上 JSON-RPC Path 中的 http 可以用 ws 替代, 代表使用 WebSocket 协议
    • 当使用 https://aria2c.com 访问时, 需要使用 https 或 wss 协议

六、BT设置 （解决Aria2 BT下载速度慢没速度的问题）

若要BT下载，还需要把BT的Trackerlist 加入设置中
Trackerlist列表地址

  1. 先修改Aria2.conf配置文件三个设置
  2. 再添加 bt-tracker= 后面加trackerlist（以英文逗号分隔开来）
    • 网上有热心网友的处理过后的列表，非常方便！自动处理了 （备用地址）
    • 或者，用 Trackerlist列表地址
      • 把list复制到 编辑器，删除空行，并且加入英文逗号分隔每行

如果你用的AriaNG的话，可以把list添加到 AriaNG – Aria2 Settings – BitTorrent Setting – option.bt-tracker.name
