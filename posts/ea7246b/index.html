WordPress淘客商城

服务器推荐安装宝塔，一键配置环境

域名和IP提前绑定好

本文以宝塔为例

一、下载安装Wordpress

可参考：官方教程

1.下载到服务器

  1. SSH登陆到服务器，到网站目录下：cd /www/wwwroot

  2. 到官网（简体中文版网站），复制 下载 .tar.gz 的链接

  3. 下载tar包，ssh中输入命令：wget 链接

  4. 解压到本地：tar -xzvf latest-zh_CN.tar.gz

（当然，你也可以下载到本地电脑，然后ftp上传，或者宝塔直接安装）

2. 新建站点

  1. 宝塔 - 网站 - 添加站点

  2. 填写信息，提交

    • 域名
    • 根目录：可以自己改个名字
    • FTP
    • 数据库

  3. 记下保存下来的信息

  4. 进入刚刚解压的wordpress文件夹中，全选，剪切，到站点更目录中 粘贴所有

    • 为什么不用解压的 wordpress文件夹？ 可能会出现权限问题，所以干脆让宝塔建立

      比如出现 需要访问您网页服务器的权限的解决办法 问题

3. 配置数据库（几乎都在宝塔中操作）

  1. 给服务器安装 phpMyAdmin

  2. 给服务器添加数据库 （数据库–添加数据库-填写信息，记下信息）

  3. 宝塔 - 文件--到wordpress目录下

  4. 把 wp-config-sample.php重命名为 wp-config.php，并编辑后保存

4. 开始安装

  1. 如果你的域名已经绑定好了，直接访问域名即可自动跳转到安装配置

  2. 如果没用绑定好，

    • 选择 - 站点 - 点击默认站点 –选择你的站点，把IP都指向这个网站，提交
    • 输入你的IP或者域名，进入配置
    • 即会自动调转到 域名或者IP/wp-admin/install.php

  3. 填写信息，安装完成。
