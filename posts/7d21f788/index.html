1.系统激活

2.系统启动项设置

Q1：Win10启动项的文件夹在哪里

A1：分为两种启动项文件夹：

  • 用户启动文件夹：Win+R - 运行中输入 - shell:startup,自动打开启动项文件夹
    - `C:\Users\Administrator\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup`
  • 系统启动文件夹：Win+R - 运行中输入 - %programdata%\Microsoft\Windows\Start Menu\Programs\Startup
    - `C:\ProgramData\Microsoft\Windows\Start Menu\Programs\Startup`
  • 用户和系统启动文件夹的区别：（在于多个用户时会有明显区别和影响）
    - 系统级别的，就是全局，所有用户通用
   - 用户级别的，就只是某个用户有此启动项

3.右键菜单 - 点击文件效果

技巧1：通过“XX”打开文件（也就是此文件以某种方式打开）

  1. 运行–regedit – 到计算机\HKEY_CLASSES_ROOT\*\shell处

  2. 右键shell - 新建 - 项 （也是文件夹）软件名，此新建的项下的内容如下 （没有的可以右键-新建-选择类型）

    • 默认项
      • 名称：默认
      • 类型：REG_EXPAND_SZ(可扩充字符串)
      • 数据：你想要显示的菜单名 （比如： 通过Code打开）
    • Icon (可选)
      • 名称：Icon
      • 类型：REG_EXPAND_SZ(可扩充字符串)
      • 数据：你想要显示图标的路径 （比如： D:\Setup\Microsoft VS Code\Code.exe）

  3. 在此项（文件夹）下再次右键 - 新建 - 项

    • command （发送给某软件打开的名字）

      • 名称：默认

      • 类型：REG_EXPAND_SZ(可扩充字符串)

      • 数据：你想要发送给XX软件打开的路径 + 命令%1 （比如： "D:\Setup\Microsoft VS Code\Code.exe" "%1" ，改下路径即可。注意，需要加 ""）

4.右键菜单 - 点击空白处效果

技巧1：点击空白处，在此处文件夹打开XXX（软件定位到此文件夹）

  1. 运行–regedit – 到计算机\HKEY_CLASSES_ROOT\Directory\Background\shell处

  2. 右键shell - 新建 - 项 （也是文件夹）软件名，此新建的项下的内容如下 （没有的可以右键-新建-选择类型）

    • 默认项
      • 名称：默认
      • 类型：REG_SZ(字符串值)
      • 数据：你想要显示的菜单名 （比如： Git GUI Here）
    • Icon (可选)
      • 名称：Icon
      • 类型：REG_SZ(字符串值)
      • 数据：你想要显示图标的路径 （比如： C:\Program Files\Git\cmd\git-gui.exe）

  1. 在此项（文件夹）下再次右键 - 新建 - 项

    • command （发送给某软件打开的名字）

      • 名称：默认

      • 类型：REG_SZ(字符串值)

      • 数据：你想要打开的XX软件的路径 + 命令%v （比如： "D:\Setup\Microsoft VS Code\Code.exe" "%V" ，改下路径即可。注意，需要加 ""）

技巧2：[CMD 或 Powershell ]桌面或者文件夹空白处，按住Shift点击鼠标右键时，将在此处打开命令改为CMD或Powershell

方法1：
1. 注册表（regedit）定位到 `计算机\HKEY_CLASSES_ROOT\Directory\Background\shell\cmd`
2. 默认有项名称为HideBaseOnVelocityId
    - 将其修改为ShowBaseOnVelocityId,就是CMD（`在此处打开命令窗口`）
    - 修改未HideBaseOnVelocityId,则是`Powershell`
3. 如果出现 “重命名值时出现错误”，则是权限问题
    - 注册表的CMD文件夹，右键--权限--高级--最上方的，所有者，更改--添加你的当前用户就可以了

5. 输入法

Q1：Win10游戏时，会打出输入法，怎么办

A1：

1. 右键右下角输入法-`区域和语言`-`添加语言+号`-`英语`，安装-把英语输入法调到最上面
2. 右侧-`高级键盘设置`-`语言栏选项`-`高级键设置`-点击，在输入语言之间-`更改按键顺序`-左边的，切换输入语言选择，未分配，确定
3. 以后用`win＋空格`切换输入法

三、笔记本

Q1.笔记本连接蓝牙耳机会卡顿，怎么办？（换了多个蓝牙耳机都是一样）

本人电脑：华为笔记本，无网口、2个USB

A1：由于蓝牙和网卡公用，连接无线的时候，造成蓝牙模块问题（负载？）

解决办法：用有线连接笔记本，笔记本无网卡，就买个USB转RJ45网卡。成功解决！
