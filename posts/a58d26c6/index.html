开源私有云盘-NextCloud （WebDAV）

一、简介

Nextcloud是一款开源免费的私有云存储网盘项目，可以让你快速便捷地搭建一套属于自己或团队的云同步网盘，从而实现跨平台跨设备文件同步、共享、版本控制、团队协作等功能。它的客户端覆盖了Windows、Mac、Android、iOS、Linux 等各种平台，也提供了网页端以及 WebDAV接口，所以你几乎可以在各种设备上方便地访问你的云盘。

官网

Github

第三方应用插件App下载

二、安装到服务器中

两种安装方式：

  1. 通过Web Installer安装
  1. Docker方式安装

Web Installer方式

官网教程

基本流程：安装LNMP运行环境 –> 通过Web Installer安装 （因为是PHP写的，所以需要安装环境）

  1. 利用宝塔安装LNMP运行环境
  2. 宝塔建立网站，填写域名、创建数据库

setup-nextcloud.php方式

  1. 下载官网安装文件.php，然后将文件保存到计算机
  2. 将setup-nextcloud.php上传到你的网站根目录
  3. 浏览器访问：域名/setup-nextcloud.php
  4. 根据要求填写内容，自动安装

zip压缩包方式

如果用 setup-nextcloud.php 安装很慢，可以下载安装的压缩文件，再上传到服务器中安装

利用宝塔解压下来，再把所有文件剪切出到网站根目录

直接访问网站域名，然后根据要求填写内容，自动安装

Docker方式

官方Dokcer地址

基本流程是：安装Docker并启动 –> 运行Nextcloud容器 –> 访问Web端初始化。

访问http://{你的服务器ip地址}:{端口}

参考：

利用Nextcloud搭建私有同步云盘

三、安装插件App

以BookMarks插件为例

  1. 从官网下载app
  2. 根据nextcloud的版本选择压缩包，下载到服务器中
    • 可以下载到本地，再上传到服务器
    • 也可以服务器中直接 wget 下载地址 （建议先到apps文件夹中，再下载）
  3. 解压到NextCloud的apps文件夹中 tar -zvxf <file-name>
  4. 到你的后台，找到插件管理页面，选择启用插件即可
