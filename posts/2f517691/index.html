python之注册为Windows开机启动服务

几个常用的服务命令

均在管理员条件下运行才能生效

  1. sc delete xxx 删除服务
  2. sc start xxx 开启服务
  3. sc stop xxx 停止服务
  4. sc query xxx查询服务的状态和pid
  5. taskkill /PID XXX /F 根据pid强制删除服务（当服务卡住不能删除时）

一、自写注册服务程序

1、说明

安装：

  1. python3.6 32位 （3.5以上均可 64位也可，具体看每个人的需求）
  2. pyinstaller(pip install pyinstaller)
  3. pywin32(pip install pywin32)
  4. flask(pip install flask)
  5. flask_cors(pip install flask_cors)

2、代码

main

flask_test

3、运行

3.1 本机（有python环境的电脑，即不打包使用）使用方法

  1. 以管理员身份运行cmd到项目目录，
  2. 执行main.py install，服务已注册
  3. 到服务中，启动它
  4. 在本地电脑的浏览器上访问地址查看：http://127.0.0.1:8080/hello

3.2 打包成exe后注册服务

  1. 把py文件打包成exe：

    • pyinstaller -F main.py
    • 也可以用pyinstaller -F -w xxx.py，无控制台版
      • 计算机管理 - 任务计划程序 - 设置开机启动任务

  2. 把main.exe拷贝出来

  3. 执行 main.exe install 安装服务

  4. 或者用bat方法,把里面的内容 ：换成自己的项目地址和项目文件

    • 保存为 start_service.bat
    • 执行它，运行完即可在服务里看到我们的服务已经注册并启动了

4、坑

  1. pyinstaller不能成功打包使用FastAPI框架，所以换成flask框架

  2. 服务无法启动

  3. 服务启动瞬间停止

  4. 打包后的exe文件不能注册为服务，提示 Failed to execute script main,但是程序运行又是正常的

  5. 如果服务出现问题，可以到系统日志中查看原因

    • 计算机管理 - 系统工具 - 事件查看器 - Windows日志 - 应用程序

二、使用nssm

  1. 下载解压缩：地址

  2. 将文件夹下的nssm.exe复制出来

  3. cmd下执行命令：nssm install name (这个名称你可以自己定义)

  4. 然后会弹出来一个界面

  5. 填完后直接点Install service就好了

  6. 查看系统服务/配置自启动

    • win+R运行 services.msc，打开系统服务
    • 找到我们刚才注册的服务，如图，修改为自动启动，然后点启动

  7. 如果我们要删除服务，那么将第四步的操作改为： nssm remove <服务名称>
