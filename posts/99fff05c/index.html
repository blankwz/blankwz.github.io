本站点Hexo配置

主题 - Butterfly

第一次建Hexo站，感谢作者！教程文档写的很清楚！让新手少走很多弯路

  • 官方Github
  • 作者站点
  • 文档

安装流程：

  1. 安装Dev分支 - git clone -b dev https://github.com/jerryc127/hexo-theme-butterfly.git themes/Butterfly
  2. 安装pug 以及 stylus 的渲染器 - npm install hexo-renderer-pug hexo-renderer-stylus --save
  3. 字数统计插件 - npm install hexo-wordcount --save
  4. 文章置顶插件 - npm uninstall hexo-generator-index --save
  5. 在source/_data/创建butterfly.yml主题文件，并根据自己的需求更改（参照教程文档）
  6. 根据需求创建新页面：
    • hexo new page 页面名
  7. 根据需求更改导航菜单

新增其他插件

hexo-blog-encrypt - 文章加密插件

详见： Hexo - 初玩加密插件

hexo-abbrlink - 中文链接优化，固定链接

  • Github
  • 安装 : npm install hexo-abbrlink --save
  • 修改根目录的config.yml

    • permalink: posts/:abbrlink/

    • 加入 
      abbrlink:
  alg: crc32  #算法 support crc16(default) and crc32
  rep: hex    #进制 support dec(default) and hex
