装机必装系列之快速启动神器Wox

一、介绍

关于大部分的介绍，可以看 异次元的文章，虽然是2014年写的，现在更强大了。

一款国产开源免费的软件快捷启动工具 Wox！它可以搜索你电脑上的程序，通过快捷键呼出搜索框，输入少量字母快速匹配出对应软件并启动、打开网页、翻译单词、或配合 Everything 插件搜索任意文件等等……

总结 Wox 的一些特性：

  • 可以搜索本机安装的程序，支持中文拼音模糊搜索
  • 搜索浏览器书签
  • 瞬间搜索出本地硬盘所有文件（通过 Everything 插件）
  • 网页搜索功能，可以给某个网址设置关键字例如搜索google，可以用g 搜索内容
  • 支持插件扩展
  • 支持自定义主题

无法搜索问题：

  1. 重启Wox
  2. 触发关键字后需要加空格

三、基础讲解

讲解版本 v 1.4.1196

激活Wox : Alt + Space （可以设置中更改）

搜索出结果后，可鼠标右键点击，出现更多操作

  • 管理员方式打开
  • 打开它的文件夹
  • 等等

1. 插件安装

使用wpm命令安装（推荐）

Wox内置了wpm插件，专门用于管理插件。

  1. 安装插件：wpm install <插件名字>
  2. 删除插件: wpm uninstall <插件名字>
  3. 已安装插件： wpm list

下载Wox插件包，手动安装

如果由于某种原因，用户不能使用wpm来安装插件。 那么用户可以将插件安装包下载到本地（以.wox结尾），然后将插件安装包拖拽到Wox搜索框上即可开始安装。

通过源码安装

下载插件源码，放到Wox的Plugins目录下

2. 插件使用方法

输入： 插件的触发关键字 + 空格 + 内容

  1. 触发关键字 查看方法：设置 - 插件 - 随便选中一个插件 - 插件名称下面会有个 触发关键字
    • 点击 触发关键字 可以更改 触发关键字
  2. 插件设置：部分插件需要进行设置，比如添加路径

3. 自定义查询热键

目的：通过快捷键，快速填入 字符串

  1. 你经常使用的 查询语句
    1. 打开某个网页
    2. 打开某个收藏夹：b zhihu 打开收藏夹中的知乎
    3. 打开某个文件夹
    4. 打开某个app
    5. cmd命令快速输入： > ipconfig
  2. 你经常使用的 触发关键字 ： 比如
    1. ALT +b –> bd 快速百度搜索，只要输入关键字就好

修改方法：设置 - 热键 中更改

4. 增加程序的搜索目录

有的软件搜索不到

  • 可能是关键路径没加入
  • 也有可能是文件后缀没有加入

方法：

  1. 设置 - 插件 - 程序 - 增加
  2. 索引文件后缀 - 增加后缀
  3. 重新索引

四、常用功能

其他触发关键字查看：设置 -插件 - 插件名 - 查看插件的列表（如果有的话）

1. 插件 - 网页搜索

google：g 关键字

百度：bd 关键字

Github: github 关键字

也可以自定义某个网页的搜索，

  • 对于有基础的人来说，方法很简单
  • 看案例，加入网页的搜索地址页面 + {q}即可
  • 比如 url ：https://www.zhihu.com/search?type=content&q={q} , 关键字 ：zh
    • 激活wox，输入 ： zh windows常用软件

2. 插件 - 打开收藏文件夹

  1. 先添加常用文件夹，方法：插件 - 文件夹 - 添加路径
  2. 设置触发关键字：比如 w
  3. 激活Wox后输入：触发关键字w，即可

3. 插件 - 浏览器书签

  1. 先加入浏览器的路径
  2. 激活Wox后输入：b 关键字

4. 替换Win+R

激活Wox后输入：> 命令 （有空格）
