Python笔记整理之列表

基础

  • 增：list.append('元素对象') - 列表尾添加元素

  • 插入： list.insert(下标, 元素) - 在列表的任何位置添加新元素

  • 删：

    • del list[下标] - 删除列表指定下标元素的值
      • 如果你要从列表中删除一个元素，且不再以任何方式使用它，就使用del 语句
    • list.pop(下标)- 删除（弹出）列表指定下标元素的值，并返回该下标元素的值，默认为-1
      • 如果你要在删除元素后还能继续使用它，就使用方法pop()
    • 查删 list.remove(元素对象) - 删除某个值的第一个匹配项（只删除第一个）

  • 统计：

    • len(list) - 返回列表元素的个数
    • list.count(元素对象) - 统计元素对象obj在列表中出现的次数

  • 合并：

    • lista+listb - 合并两个列表
    • lista.extend(listb) - 把列表b的元素 加到 列表a的 末尾

  • 重复：list * n - 列表的元素重复n次

  • 排序：

    • list.reverse() - 反向列表中所有元素
    • list.sort(cmp=None, key=None, reverse=False)
      • cmp – 可选参数, 如果指定了该参数会使用该参数的方法进行排序。
      • key – 主要是用来进行比较的元素，只有一个参数，具体的函数的参数就是取自于可迭代对象中，指定可迭代对象中的一个元素来进行排序。
      • reverse - reverse = True 降序， reverse = False 升序
    • sorted(iterable, cmp=None, key=None, reverse=False) - 排序，同上
      • 与list.sort()的区别：
        • sorted()返回新的列表，不改变原有列表的值;
        • list.sort()无返回值，直接对原有列表进行排序

  • 查索引：list.index(元素对象[, start[, end]]) - 从列表中找出某个值第一个匹配项的索引位置，如果没有找到对象则抛出异常

a = ['Hello']          
a.append('Google')    # 增 a:['Hello','Google']
del a[0]  #删 a:['Google']

len(a) # 统计长度 a:1
a.count('Google') #统计出现的次数 a:1

b = ['Great']
c = a + b  #合并 c:['Google','Great']
a.extend(b) # 合并 a:['Google','Great']

d = b*2 #重复 d:['Great','Great']
d.insert(0,2020) #插入 d:[2020,'Great','Great']

#查删
e = [1,2,3,4,3,5,6]
print(e.pop())  #  6
print(e.pop(0)) #  e：[2,3,4,3,5]
e.remove(3) #  e：[2,4,3,5]

截取

  • list[n] - 截取第N+1个元素
  • list[-n] - 截取倒数第N个元素
  • list[n:] - 截取从N+1个元素到最后所有元素
  • list[x:y:] - 截取从 x+1 到 y+1 的元素
