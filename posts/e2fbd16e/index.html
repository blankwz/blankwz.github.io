Windows装机及软件推荐

一、安装系统

系统下载

  1. 省事法：下载Ghost版系统，但系统不干净，可能会有后门
    • 地址：随便找吧，都差不多
  2. 洁癖法：下载纯洁版系统，麻烦在于驱动
    • 地址：MSDN的itellyou
    • 驱动问题：碰到过装完系统后，连USB的鼠标键盘都无法识别，-_-||

PE工具

  • 推荐使用微PE工具箱：点击下载

纯净系统驱动下载方法(任选其一)

  1. 用万能驱动EasyDrv7（会安装第三方软件）
  2. 先准备个网卡驱动，有网后，就可以用各种驱动大师，驱动精灵等装驱动。装完后就过河拆桥卸载

Windows运行库和.NET框架下载 - 地址

  • VC运行库下载： 其他工具和框架 - Microsoft Visual C++ Redistributable for Visual Studio 2019
  • .NET框架：其他工具和框架 - .NET Framework

二、软件推荐

装机必备

  • Chrome - 浏览器，不解释
  • Everything- 强大的Windows的文件检索工具
  • unlocker - 强力解除文件占用工具
  • ALTRun - 快速启动软件
  • CCleaner - 垃圾清理，系统优化软件
  • Clover - 资源管理器的强大扩展
  • Cryptomator - 轻松加密网盘上的文件，官网

学习

  • CopyTranslator - 复制即翻译的外文辅助阅读翻译解决方案
  • 灵格斯词霸
  • Wolfram Mathematica - 用于解决和数学、数字相关的几乎一切问题

开发

  • SFTP工具 (任选其一)
    • FileZilla
    • WinSCP
    • Cyberduck（MAC版本）
    • XFTP
  • SSH工具 (任选其一)
    • Windows内置 PowerShell
    • Putty
    • Xshell
    • FinalShell(全平台)，需要JAVA和Winpcap
    • MobaXterm(全终端，推荐！)

下载

  • PanDownload - 百度网盘资源下载神器
  • xdown - 官网下载
  • Motrix - 支持下载 HTTP、FTP、BT、磁力链、百度网盘等资源
    • 官网
    • 文档

视频图片

  • Snipaste - 强大的截图软件！强大的贴图功能，用来学习工作，好用的不要不要的

    • 官网
    • 文档
    • 其他网站的介绍

  • 维棠FLV - 视频网站解析下载

  • Ocam - 视频录制

  • Potplayer - 简洁强大的视频播放器

效率

  • Ditto - 增强剪切板，避免重复ctrl+c、ctrl+v
  • Q-dir - 多窗口文件整理工具,不同窗口文件快速切换
  • fences - 桌面管理工具

安装

  • LastActivityView - 查看电脑的最近活动
    • 官网

游戏

  • AIM HERO - FPS游戏训练，训练反应，训练瞄准软件
    • 同类软件：AIM BOOSTER，比AIM HERO的更初级。
