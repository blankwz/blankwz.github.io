跨平台浏览器「书签同步」插件 - Floccus

一、简介

Floccus 是一款可以通过 Nextcloud，WebDAV 或本地文件（以及任何文件同步解决方案）跨浏览器同步书签的插件，支持 Chrome、Firefox、Kiwi 等浏览器（不支持 iOS）

项目地址: Github – Floccus

目前 Floccus 可以使用以下同步方法：

  • WebDAV ：目前支持 「坚果云、nextcloud / owncloud，box」 等 WebDAV 服务器
  • Nextcloud 书签 ：利用 Nextcloud 自带的书签应用程序进行同步，可通过 Web 访问。(需要nextcloud v12 以上版本)。
  • 本地文件同步 ：利用 LoFloccus 4 软件将书签同步到本地, 再利用 Dropbox，Syncthing，rsync 等进行跨设备同步。

注意： 浏览器中内置的书签同步服务可能会导致同步出现问题, 建议关闭。

注意： nextcloud 中的 bookmarks_fulltextsearch 会导致其中的书签应用程序出现问题, 请不要在 nextcloud 中安装 bookmarks_fulltextsearch。

WebDAV方式

  1. 如果你能折腾，可以拥有自己的一台nextcloud / owncloud 服务器
  2. 不折腾，直接方便的用坚果云

这种方式是直接保存为一个XBEL的书签文件，类似书签导出的HTML文件

Nextcloud 书签

这种方式是利用Nextcloud 的 Bookmarks 插件保存

本地文件同步

这种方式我还没有试验

二、Chrome插件

  1. 通过Chrome网上应用店安装它

  2. 也可以通过下载最新版本的Chrome程序包并将其放入Chrome的扩展程序页面来进行安装

用法

  • 帐户面板; 安装后，将弹出“帐户”窗格，使您可以创建和管理帐户。通过单击浏览器工具栏中的絮状图标，您将始终可以访问此窗格。
  • 您的帐户：您可以设置多个帐户，并为每个帐户选择一个书签文件夹，该文件夹应与该帐户同步。每当添加或更改书签时，Floccus都会使书签与您选择的服务器保持同步，并且还会定期同步以从服务器中获取最新更改。
  • 同步根文件夹：如果要同步浏览器中的所有书签，则需要在文件夹选择器中选择最上面的无标题文件夹。（以防万一，您可以立即使用来自不同供应商的浏览器来同步根文件夹，因为现在已标准化了内置文件夹的名称）。

在执行第一次书签的同步之前，强烈建议先对现有的浏览器书签进行备份导出，以防出现意外

三、服务端 - 坚果云

坚果云是一个网盘，但坚果云是国内唯一支持 WebDAV 协议的云储存。

  1. 打开坚果云

  2. 设置第三方应用密码

    • 右上角 - 账户信息 - 安全选项
    • 添加应用 - 输入名称【Floccus】- 生成密码

  3. 在 我的坚果云文件夹里 建立一个 Floccus 文件夹

  4. 在Floccus 浏览器插件里添加账户

  5. 选择 WebDAV共享中的XBEL文件

    1. WebDAV URL：https://dav.jianguoyun.com/dav/我的坚果云
    2. 用户名：填写你坚果云的账户邮箱
    3. 密码：填写你在之前创建的【第三方应用密码】
    4. 书签路径：Floccus/bookmarks.xbel

  6. 文件夹映射与同步行为，自行选择

四、服务端 - NextCloud

XBEL文件方式

  1. 拥有一台 NextCloud 服务器
  2. 建立 一个Floccus文件夹 （可选）
  3. 在Floccus 浏览器插件里添加账户
  4. 选择 WebDAV共享中的XBEL文件
    1. WebDAV URL：你的NextCloud的WebDAV地址
    2. 用户名：NextCloud用户名
    3. 密码：NextCloud密码
    4. 书签路径：Floccus/bookmarks.xbel

NextCloud插件方式

  1. 服务器中下载Bookmarks插件
    • 先到NextCloud服务器的apps文件夹中： cd ./apps
    • wget https://github.com/nextcloud/bookmarks/releases/download/v3.4.4/bookmarks-3.4.4.tar.gz
  2. 解压 ：tar -zvxf bookmarks-3.4.4.tar.gz
  3. 到你的后台，找到插件管理页面，选择启用插件即可
