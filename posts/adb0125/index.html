最绿色，快速启动程序

一、WIN+R方式的优缺点

Win+R 是 Windows+R的缩写，Windows键位于键盘Ctrl与Alt中间
快速启动想要的软件只需两步：

  1. 快捷键：WIN+R
  2. 程序名的简称 + 回车

优点

  1. 绿色：无需安装任何软件
  2. 快捷：鼠标都不要，只需要键盘输入即可
  3. 稳定：不用任何软件，当然稳定
  4. 个性化：自己随意命名
  5. 扩展：bat文件程序了解下
  6. 隐秘性：可以让别人找不到你的程序

缺点

  1. 不直观：相比于一些快速启动软件，需要自己记忆程序名称
  2. 不美观：无任何界面
  3. 不够高效

    可能会觉得不高效，不智能，没关系，这只是个思路。

二、使用方法

1.统一管理程序的快捷方式

找到你经常用的程序，建立快捷方式，统一（复制）放在某个文件夹下（以下简称 环境变量目录），比如D:\link

建立快捷键方式：右键 - 发送到→桌面快捷方式 或 右键 -创建快捷方式

Q：可以打开哪些程序

任何可以有快捷方式的程序，都可以打开！
包括：文件夹，各种格式的程序、网页（需要加后缀 url）

2.为程序重新命名

这里的命名，就是你以后经常输入的名称，很重要！
命名的好坏关系到你自己启动程序的速度

一些建议：

  1. 使用英文或者数字，或者英文+数字。 方便书写
  2. 名称尽量短，快速键入
  3. 方便自己记忆的名称
  4. 不要与系统的程序和 其他在系统环境变量中的程序 重名 (比如：cmd，java，python等等)

3.把 环境变量目录 加入系统环境变量中

win7系统不一样，我是win10；其他系统建议搜索下，如何 把文件夹加入系统环境变量
有的

  1. 右键 → 我的电脑 → 属性 → 高级系统设置 → 环境变量
  2. 选择 用户变量 或 系统变量 (不多解释了，解释多了会晕)
  3. 找到Path，编辑 - 新建，把刚刚的 D:\link 加入进来
  4. 一路确定下去~~保存了就好了

三、一些技巧

1. 如果你不喜欢快捷Win+r，把 运行 放置在任务栏中

- win10中，点击开始图标 - 找到`windows系统-运行` - 右键 - 固定到任务栏
    - 或者直接在任务栏中的搜索  `运行` - 右键 - 固定到任务栏

2. 使用bat程序，完成一些复杂的操作

比如：快速更换IP， 看这篇

3. 可以更隐秘

- 把你的`快捷方式文件夹`藏起来，越深越好。
    - 比如藏在某个程序文件夹中（小心，万一你哪天把这个程序删掉了。。可能会消失）
- 把你的`秘密文件夹`藏起来，很深那种。。。然后你懂的。

4. 如何打开文网页

1. 加后缀运行，比如  123.url + 回车
2. 或 使用浏览器的目标值方法，
    - 多建立个chrome浏览器快捷方式，然后修改属性的目标值
    - 比如改为 "C:\Program Files (x86)\Google\Chrome\Application\chrome.exe" "www.baidu.com" 
    - 再该快捷方式名称即可
    - bd + 回车

5. 浏览器中，快速搜索引擎搜索

在Chrome中，快速谷歌搜索内容

搜索方式 ： g + 搜索内容 + 回车

  • 环境变量目录下建立 g.bat ，输入以下内容，保存

    可以替换为其他搜索引擎地址，自己替换
    也可以自己替换其他浏览器，但是确保其他浏览器也在环境变量中
