利用腾讯COS制作个人图床

COS:对象存储（Cloud Object Storage，COS）

参考：这

官方教程

一、环境搭建

  • 安装 nodejs （LTS版本就好）

    • 在windows环境变量中增加一项NODE_PATH，最好同时指向2处，例如：C:\Users\ZNMLR\node_modules和C:\Users\ZNMLR\AppData\Roaming\npm\node_modules最少指向一处
    • 前者对应npm的本地安装，后者对应npm的全局安装

  • 腾讯云SDK

    1. 安装SDK cos-nodejs-sdk-v5 ：运行CMD - npm i cos-nodejs-sdk-v5 --save（需要在C盘中安装）
    2. 打开腾讯云SDK，下载源码，解压
    3. 到解压目录下，SHfit+右键 - 此处cmd（或者powershell）打开
    4. 命令：npm install -g

二、腾讯云注册并且创建

  1. 请到腾讯云注册账号并实名认证
  2. 在对象存储 中 —- 创建存储桶（记得将存储桶访问权限设置为共有读私有写，否则后面会因为权限不足导致返回的链接失效）
  3. 打开刚刚创建的桶---设置，选择域名管理，打开默认加速域名（编辑–打开 当前状态）
  4. CND服务 — CDN控制台–开通，
  5. 回到刚刚的 域名管理— 点击列表中的域名— 高级配置
  6. 打开 回源鉴权 — 保存
  7. 高级配置中的HTTP Header配置，打开并添加HTTP Header。
    • 参数选择Content-Disposition,
    • 取值写inline;filename=FileName.txt
  8. 回到 对象存储 复制存储桶名称、所属地域（括号里面的英文，ap-xx）
  9. 登录 访问管理控制台 ，新建密钥 获取APPID、SecretId、SecretKey

三、Windwos使用（CMD与Node JS）

操作方法：在任意文件上点击右键，选择 上传腾讯云 会有脚本框一闪而过，找个文本框粘贴下，就得到你想要的URL+

在图片上右键上传，自动生成图片URL到剪切板中，任意地方Ctrl+v均可粘贴

在文件上右键上传，自动生成图片URL到剪切板中，任意地方Ctrl+v均可粘贴

原理：通过js脚本上传到服务器，并且把js脚本放置到bat中执行，再把bat文件添加到右键（注册表修改）

  1. 注册表中添加bat （地址自行更改）

    • 运行–regedit – 到计算机\HKEY_CLASSES_ROOT\*\shell处
    • 新建项上传腾讯云，在此项下再次新建项command，
    • 修改右侧默认值 cmd.exe /K " C:\cos\cos.bat "%1"" （路径改为你刚刚新建的bat文件地址）
