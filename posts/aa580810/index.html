把机器人加入群组方法

机器人加入群组后，可以帮忙管理群组，一个群组可以有多个机器人

  1. 点击机器人名字
  2. 最下面，ADD To Group，选择群组即可（需要有群组权限）

二、写代码

如果需要增加/命令，记得先在 BotFather 中 添加下命令（Edit Commands）

0 工具

  • 谷歌工具：google script console，以下简称 GSC
  • Mango DB数据库
    • mLab 线上免费数据库，通过它的API进行存储

1 第一次简单使用

借用工具：google script console，以下简称 GSC

  1. 进入google script console ，点击 start Scripting

  2. 新建项目，点击名称，起个名

  3. 书写代码 (更改 代码.gs 为下面峰哥的代码)

    • 注意 UrlFetchApp.fetch 后地址的格式：
    • https://api.telegram.org/bot+ 你的Token

  4. 部署

    • 旧版本：点击 发布 - 部署为网络应用
    • 新版本：右上角 - 部署 - 新建部署
      • 选择类型：Web应用
      • 说明：随便
      • 访问权限人员：任何人
    • 点击部署
    • 点击访问权限，选择你的谷歌账户，点击允许
    • 生成了你的应用的地址，保存它

  5. 生成 关联链接 （需要把你的应用地址发送给TG）

    • Set Webhook 的链接格式：https://api.telegram.org/bot + 你的机器人Token +/setWebhook?url=谷歌应用地址
    • 需要Encode下谷歌应用地址！！！
      • 站长工具
      • 或者 其他

  6. 浏览器中打开上面的链接，如果结果是 {"ok":true,"result":true,"description":"Webhook was set"},说明成功

2 后续更新代码

  1. 在 google script console 中更新代码，保存
  2. 发布 （不需要重新关联链接，GSC会自动更新版本）
    1. 旧版本GSC： 部署为网络应用 - 项目版本 - 新建个 ，更新，确定
    2. 新版本GSC：右上角 - 部署 - 新建部署 - 输入新描述，确定即可
  3. 完成！（记得在BotFather中添加相应的命令）

三、说明与格式

格式

  • Set Webhook 的链接格式：https://api.telegram.org/bot + 你的机器人Token +/setWebhook?url=应用地址（应用地址需要Encode）

官方说明

  • 信息分析，比如：
    • 新进群人员信息：new_chat_members
    • 离群人员信息：left_chat_member
  • 用户信息
