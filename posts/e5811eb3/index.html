一、初级 HM NIS EDIT

官网：地址

参考：方便快捷的客户端打包工具“HM NIS Edit”

很简单，安装流程来做即可

  1. 文件 - 新建文件 - 向导，下一步
  2. 修改程序信息，下一步
  3. 程序选项，下一步
    1. 选择安装程序图标（可忽略）
    2. 安装程序文件：安装程序保存在哪
    3. 选择 SimpleChinese
  4. 目录和授权文件，下一步
    1. 目录，可以默认
    2. 授权文件，随便建个txt文件，上面写上 授权信息（可以随便写）
  5. 指定程序文件，下一步
    1. 删除右边的两个文件
    2. 按钮AddDirTree - 选择你的安装程序整个目录
  6. 图标和快捷方式，默认，下一步
  7. 安装后运行，也默认，随意，下一步
  8. 解除安装程序，也默认，随意，下一步
  9. 可以勾选脚本，完成，选择保存脚本的路径
    • 脚本可用于以后更新程序后，重新打包用
  10. 生成脚本后，编译脚本，Ctrl + F9
  11. 等待完成，找到刚刚保存的安装程序，试验下。

二、NSIS: Nullsoft Scriptable Install System

官方下载：地址

汉化增加版下载：地址

NSIS 是“Nullsoft 脚本安装系统”(Nullsoft Scriptable Installation System) 的缩写，它是一个免费的 Win32 安装、卸载系统，采用了简洁高效的脚本方式。它本来是一个由 Nullsoft 创建并用于作为 Winamp 及其插件发布的系统，但现已被上百个应用程序所应用，作为它们进行程序发布的工具。

NSIS 创建的安装程序可以进行安装、卸载、设置系统设置、解压文件等等。几乎可以做所有事情。因为它基于脚本文件，所以您可以完全控制您的安装程序的每个部分。它的脚本语言支持变量、函数、字串处理，就像是一个普通的程序语言 - 但其仅设计用来创建安装程序。

详细方法，还没有试，可以用汉化增加版，里面有很多教程
