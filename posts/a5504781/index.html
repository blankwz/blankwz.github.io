好物推荐之Chrome插件

安装方法：

  • Chrome网上商店直接下载安装
  • 下载离线包安装
    • 镜像站
    • 开发者主页
    • 第三方站

1、我的必备

  • Tampermonkey - 油猴，神器！最强插件，不解释！
    • 油猴脚本下载：
      • 普通站
      • 成人站
    • 脚本推荐
      • My Novel Reader 小说阅读脚本，统一阅读样式，内容去广告等
      • 挊 自动获取磁链接并自动离线下载
      • Easy offline 一键自动将磁链、bt种子或其他下载资源离线下载至网盘

  • AdBlock - 最佳广告拦截工具
  • Proxy SwitchyOmega - 代理软件，梯子必备

  • SimpleUndoClose - 撤回最近关闭的网页

  • 扩展管理器（Extension Manager）- 一键管理所有扩展，快速开启/禁用

  • Neater Bookmarks - 收藏夹快速检索

  • Evernote Web Clipper - 印象笔记用户必备

  • 划词翻译 - 便捷翻译

  • FireShot 捕捉网页截图

  • Video Downloader professional - 视频嗅探下载

  • 链接批量复制助手 - 方便复制页面内大量的ed2k、magnet、迅雷、快车、QQ等链接的应用

  • Enable Copy - 解除右键及复制粘贴限制

  • Quickey Launcher - 快捷键打开网址

  • Search the current site (站内搜索)

  • Fatkun - 图片批量下载

  • Bookmarks Checker - 书签坏链检查

  • Stylish-为任意网站自定义主题

  • Vimium - 用vim方式控制浏览器

  • Momentum - 推荐！漂亮的标签页，格言等

  • Toby - 存储没看完的标签，一键管理

    • 同类插件：OneTab Plus

2、Github 插件

  • Octotree - Github上显示树形结构

  • Awesome Autocomplete for GitHub - 加强Github搜索框

  • File Icon for GitHub and GitLab - 给Github文件夹加图标

  • Octohint - Github代码智能高亮

  • Sourcegraph - 将Github变成IDE一样强

  • Wide-Github - 让Github支持宽屏

    • 官网Github

  • Enhanced Github - 可以显示每个代码文件大小以及下载链接

    • 官网Github

  • GitHub Hovercard - 增强了 GitHub 的交互体验,在你查看头像、issue、repo 都时候都会弹出一个悬浮卡片，用以显示更多信息

  • notifier-for-github - 协作时神器！

    • 官网Github

  • github-file-icon - 给Github文件加图标

    • 官网Github

3、开发者

  • Library Sniffer - 探测当前网页所使用的类库、框架和服务器环境，为开发者提供了方便

  • XPath Helper - 提取XPath

  • Web Scraper - 爬虫工具

  • EditThisCookie - cookie管理器。添加，删除，编辑，搜索，锁定和屏蔽cookies！

  • Page Ruler Redux - Web开发人员\设计器标尺

  • Alexa Traffic Rank - 官方Alexa流量排名

  • JSONView - JSON格式排版

  • Postman - 强大的网页调试工具

  • Wappalyzer - 分析网站的技术框架

  • FeHelper(前端助手)

4、学习

  • Grammarly - 英语语法检测

  • 沙拉查词 - 强大的翻译，英语查词

  • Block Site - 设置时间段，禁止浏览某些网站，学会自律

  • Trackr - 时间去哪儿了，分配好你的时间

  • Oald 7 牛津高阶第七版 - 离线词典

5、其他

  • crxMouse Chrome™ 手势 - 通过鼠标手势操作网页

  • 购物党自动比价工具

    • 115 网盘助手 - 导出aria2-rpc的下载链接
    • 下载：Github主页

  • AutoPagerize - 自动加载分页

  • Dark Reader - 黑色主题，护眼

  • Loom for Chrome - 录屏插件

  • bilibili-helper - b站助手，下载视频等

    • 下载：Github主页

  • 简易浮动视频 - 让视频浮动出来

  • Pushbullet - 跨平台推送信息

  • FasTube - 更快的方式看YouTube视频
    • 壹伴 - 好用的微信编辑器
    • 下载：官网

  • Imagus - 鼠标停在网页图片上就可以在所在页面看到大图

  • Super History Cleaner - 清除历史记录，可以设置时间周期

  • IE Tab - 让Chrome切换到IE内核，网银专用

  • 书签侧边栏 - 书签侧边栏方式显示

  • Save to Pocket - 稍后阅读

  • Markdown Here - 让大部分网站的编辑器支持Markdown（转换）

  • iMacros - 网页自动化插件
