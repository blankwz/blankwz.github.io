OneNote使用及技巧

一、导入与转换

1.印象笔记导入OneNote

  • 印象笔记全部导入到OneNote：官方软件
  • 这个软件是直接把印象笔记导入到网页端，本地好像没有
  • 发现一个问题，导入时，竟然不要印象笔记的密码！！！这么说的话！如果想看别人资料…靠!

2.PDF 插入到OneNote

  • 打开PDF - 打印 - 发送到OneNote
  • 注意：此方法导出的都是图片格式

3.网页插入到OneNote中

  • 方法一：打印 - 发送到OneNote

    • 注意：此方法导出的都是图片格式

  • 方法二：网页剪藏软件（有官方的和非官方的）

    • OneNote clipper

4. Markdown导入OneNote

稍微繁琐点

其实就是Word和Markdown之间互相转换！

因为Word格式可以之间转换到OneNote

4.1 用Writage转换

这个软件可以让Word和Markdown之间互相转换

  • 下载

  • md转word方法：

    • word打开md文件
    • 然后另存为word

  • word转md方法：

    • 直接另存为md格式即可

  • 可能会出现的问题：

    • 图片路径问题

4.2 用Typro转换

只能MarkDown转换成Word

  1. 用Typora打开MarkDown文件 - 导出 - word
  2. 提示你安装Pandoc,安装它（强大的工具！）
  3. 重启Typora，再次打开md文件 - 导出 -Word
  4. 保存后的Word内容 - 复制
  5. 直接粘贴到OneNote中即可。

4.3 也可以直接用Pandoc转换

  • 教程
  • 把hello.md 输出成1.doc
    • pandoc -f markdown -t docx -o 1.doc hello.md
  • 或者 把hello.md 输出成2.doc
    • pandoc hello.md -o 2.docx

N.其他格式插入

  • 借助数字珍宝的批量处理器：OneNote Batch
  • 收费 ¥100.00

2.代码高亮 - NoteHighlight2016

使用方法：

  1. 下载安装
  2. OneNote - 文件 - 选项 - 加载项
  3. 最下方 （管理 COM加载项） - 转到
  4. 弹出框体 - 勾选 NoteHighLightAddln - 确定
  5. 重启OneNote！
  6. 重启后在帮助的右边会出现NoteHighLight
