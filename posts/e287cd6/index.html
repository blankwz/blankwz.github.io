    JAVA教程二之基本程序结构

    0.编译与运行

    1. 先编译 javac Welcome.java
    2. 再运行 java Welcome (文件名无需带后缀！！)

    1.区分大小写！

    2.命名

    2.1 基本

    • 不能用Java关键字命名
    • 以字母开头，后面跟字母和数字（类名、变量名）
    • 常量则全是大写

    2.2 类名：以大写字母开头

    2.3 多个单词：骆驼命名法，例 CamelCase

    3.源代码文件命名

    必须与公共类的名字相同，并用.java做后缀

    4.完整的语句后需加 ; 空白符会被JAVA编译器忽略

    所以，随便怎么分隔语句，除非遇到了 ；

    5.main方法

    类的源文件，必须包含一个main方法

    每个Java程序，都必须有一个main方法

    main方法必须是静态

    6.注释

    7.数据类型

    8种数据类型

    • 4种整型
    • 2种浮点型、
    • 1种表示Unicode编码的字符单元的字符类型Char
    • 1种表示真值的布尔类型

    7.1 前后缀

    • 长整型：后缀带L

    • 浮点型：不带F后缀的小数，默认为double

      • float：后缀F
      • double：后缀D

    • 八进制：前缀0

    • 二进制：前缀 0b

    • 数字字面量：下划线方便阅读 ，编译器会忽略下划线

    • 例如： 1_000_000 表示100万

      • Unicode编码：\u 前缀

    7.2 整型

    类型 存储需求 备注
    int 4字节 常用，范围：正负21.478…亿
    short 2字节 范围：负3.2768万 到正 3.2767万
    long 8字节 后缀带L
    byte 1字节 范围：-128到127

    7.3 浮点型

    类型 存储需求 备注
    float 4字节
    double 8字节 大部分使用double

    金融计算时，不要用浮点数值，要用 BigDecimal类

    异常

    表示溢出和出差的特殊的浮点数值

    • 正或负无穷大：例如，正整数除以0 的结果为正无穷大，浮点数被0除也是无穷大或NaN
    • NaN：例如 0/0 或者负数的平方根结果为 NaN

    7.4 Char类型：单个字符，字符常量（不建议使用）

    Unicode编码 用 十六进制： 范围 \u0000 到 \Uffff

    7.5 boolean 类型（布尔）

    • false

    • true

    8.变量

    8.1 变量先声明

    一行可以声明多个变量 (不建议)

    • int i,j;

    8.2 变量使用前需先初始化

    否则，报错！

    8.3 常量 （final表示）

    • 常量只能被赋值一次！
    • 全部大写
    类常量
    • 定义再main方法之外
    • 同一个类中其他方法都能用
    • 若是public，则其他类也能使用

    9.运算符

    9.1 二元运算符

    运算符放在赋值号的左侧

    例如 x+=4 等价于 x = x+4

    9.2 自增或自减运算符

    对数值变量的 加1 、减1 操作

    • 前缀 ++ ：先加1，再运算
    • 后缀++：先运算，再加1

    9.3 关系运算符和布尔运算符

    != ：不等于

    <=：小于等于

    >=：大于等于

    &&： 与

    ||：或

    ！：非

    ?:三元操作符

    - `condition ? expression1:expression2​`
- 当条件为真时，计算第一个表达式，否则计算第二个表达式

    && 和 || 是按照短路 方式求值：如果第一个操作数已经能确定表达式的值，第二个操作数就不必计算了

    注意： & 和 | 不是按短路方式计算

    9.4 位运算符

    &：与

    |：或

    ^：异或

    ~：非

    >>：二进制位右移，有符号位填充高位

    <<：二进制位左移

    >>> ：二进制位右移，用 0 填充高位

    没有<<<

    10.类型转换

    10.1 数值类型转换

    无信息丢失的转换(从左到右)

    • byte ---> short ---> int ---> long

    • char---> int

    • int---> double

    可能有精度损失的转换

    • int ---> float
    • long ---> float
    • long ---> double

    两个数值进行二元操作时

    1. 若一个操作是double,则另一个转换为double
    2. 否则，一个操作是float,则另一个转换为float
    3. 否则，一个操作是long,则另一个转换为long
    4. 否则，都转换为int

    也就是说按优先级，double > float > long > int

    10.2 强制类型转换

    不管丢不丢是精度或信息，我都要转换

    语法：圆括号中是给出目标类型，后面跟上待转换的变量名

    12.字符串

    1. Java没有内置的字符串类型，而是JAVA类库提供的一个预定义类，String

    2. JAVA字符串是由Unicode组成的序列，比如JAVA是 J、A、V、A组成

    12.1 方法–>substring子串

    12.2 拼接

    使用 + 号拼接

    12.3 方法–>equals 检测两个字符串是否相等（区分或不区分大小写）

    不能用 == 来判断字符串是否相等！！！！ ==只是判断两个字符串的位置是否相等

    equals ： 区分大小写

    equalsIgnoreCase：不区分大小写

    12.4 空串与Null串 （如何判断）

    空串：”” (长度为0的字符串)

    Null串：null

    12.5 代码点和代码单元（不太懂）

    常用Unicode字符用一个代码单元就可以表示，而辅助字符需要一对代码点表示

    length() :返回 字符串所需要的代码单元数量

    codePointCount()：得到实际的长度，即代码点数量

    charAt(n)：返回位置n的代码单元

    13. 字符输入输出

    • 输出：system.out.print()

      • 自动换行输出 system.out.println()
      • 格式化输出 system.out.printf() + % 格式化转换符！

    • 输入：先构造Scanner（需导入util包），关联System.in，再输入

      • import java.util.*
      • Scanner in = new Scanner(System.in)`
      • in.nextLine() # nextLine方法输入一行
        • 读取一个单词（若输入的这一行里有空格，分割了单词） ： in.next()
        • 读取一个整数：in.nextInt()
        • 读取double： in.nextDouble()

    • 密码输入：用Console

    c:\\\windows\\1.txet

    14.2 写入文件（输出）PrintWrite对象

    15.控制流程

    15.1 块

    • 块（即复合语句）是指由一对大括号括起来的若干条简单的 Java语句。

    • 块确定了变量的作用域。

    • 块可以嵌套在另一个块中

    • 不能在嵌套的两个块中声明同名的变量

    15.2 条件判断 if

    • 条件语句格式：

      • if (condition) statement
      • if (condition) statement else statement

    • 条件必须用括号括起来。

    • 加入大括号，块后，statement 可以使用多条语句

    • else 子句与最邻近的 if 构成一组

      • if (x <= 0) if (x == 0) sign = 0; else sign = -1 ;
      • 或者用大括号更清晰 if (x <= 0) { if (x == 0) sign = 0; else sign = -1 ; }

    15.3 循环 While

    判断后，再循环：while {condition) statement

    执行一次后，再判断循环：do statement while {condition);

    15.4 确定循环 for

    for 循环语句是支持迭代的一种通用结构， 利用每次迭代之后更新的计数器或类似的变量 来控制迭代次数。

    for语句：例：for (int i ;i>0 ;i--)

    • 第 1 部分通常用于对计数器初始化；

    • 第 2 部分给出每次新一轮循环执行前要检 测的循环条件；

    • 第 3 部分指示如何更新计数器。

    for 循环语句只不过是 while 循环的一种简化形

    还有一种for each循环，在数组中讲解

    15.5 多重选择 Switch （少用）

    在处理多个选项时， 使用 if/else结构显得有些笨拙，这时，就用Switch

    • switch语句将从与选项值相匹配的 case标签处开始执行直到遇到 break语句，

    • 或者执行到 switchi吾句的结束处为止。

    • 没有相匹配的 case 标签， 而有 default 子句， 就执行这个子句

    case标签可以是：

    • 类型为 char、byte、short 或 int 的常量表达式。

    • 枚举常量。

    • 可以是字符串字面量。

    15.6 中断流程控制

    GOTO：不推荐

    continue： 跳出当前循环

    - 还有一种带标签的 continue 语句，将跳到与标签匹配的循环首部

    Break：跳出循环，for while

    • 还有带标签的Break语句：

      • 标签必须放在希望跳出的最外 层循环之前， 并且必须紧跟一个冒号
      • 只能跳出语句块， 而不能跳入语句块。

    16. 大数值

    如果基本的整数和浮点数精度不能够满足需求， 那么可以使用java.math 包中的两个 很有用的类：Biglnteger 和 BigDecimaL ，可以处理包含任意长度数字序列的数值。

    • Biglnteger类实现了任意精度的整数运算

    • BigDecimal 类实现了任意精度的浮点数运算

    17.数组

    17.1 说明

    • 数组是一种数据结构， 用来存储同一类型值的集合。
    • 在声明数组变量时， 需要指出数组类型 （数据元素类型紧跟 []) 和数组变量的名字
    • 使用 new 运算 符创建数组。
      • new int[n] 会创建 一个长度为 n 的数组。
    • 允许数组长度为0，
      • 匿名数组方式书写：a = new elementType[0]

    数组（初始化时）的默认值

    • 创建一个数字数组时， 所有元素都初始化为 0
    • boolean 数组的元素会初始化为 false
    • 对象数组的元素则初始化为一个特殊值 null

    一旦创建了数组， 就不能再改变它的大小（尽管可以改变每一个数组元素）

    如果经常需 要在运行过程中扩展数组的大小，就应该使用另一种数据结构— —数组列表（array list)

    17.2 for each循环

    • 可以用来依次处理数组中的每个元素（其他类型的元 素集合亦可）而不必为指定下标值而分心

    • for each 循环语句显得更加简洁、更不易出错

    格式：for (variable : collection) statement

    定义一个变量用于暂存集合中的每一个元素， 并执行相应的语句（当然，也可以是语句块)。 collection 这一集合表达式必须是一个数组或者是一个实现了 Iterable 接口的类对象

    有个更加简单的方式打印数组中的所有值， 即利用 Arrays 类的 toString 方法

    System.out.println(Arrays.toString(a));

    17.4 匿名数组

    以在不创建新变量的情况下重新初始化一个数组

    17.5 数组拷贝

    引用到同一个数组（引用到同一个地址）

    17.7 数组方法

    sort() 排序

    static void sort(type[ ] a)

    采用优化的快速排序算法对数组进行排序。

    参数：a 类型为 int、long、short、char、byte、boolean、float 或 double 的数组

    copyOf() 或者 copyOfRange() 拷贝数组

    static type copyOf(type[]a, int length)

    static type copyOfRange(type[]a, int start, int end)

    返回与 a 类型相同的一个数组， 其长度为 length 或者 end-start， 数组元素为 a 的值

    参数：

    • a 类型为 int、 long、short、char、byte、boolean、float 或 double 的数组。
    • start 起始下标（包含这个值）
    • end 终止下标（不包含这个值）。这个值可能大于 a.length。在这种情况 下，结果为 0 或 false
    • length 拷贝的数据元素长度。 如果 length 值大于 a.length， 结果为 0 或 false ; 否则， 数组中只有前面 length 个数据元素的拷贝值

    binarySearch() 查找值

    static int binarySearch(type[] a, type v)

    static int binarySearch(type[]a, int start, int end, type v)

    采用二分搜索算法查找值 v。

    如果查找成功， 则返回相应的下标值； 否则， 返回一个 负数值 r。-r-1 是为保持 a 有序 v 应插入的位置。

    参数：

    • a 类型为 int、 long、short、 char、 byte、boolean、float 或 double 的有 序数组。

    • start 起始下标（包含这个值）

    • end 终止下标（不包含这个值)

    • v 同 a 的数据元素类型相同的值

    fill() 设置为相同的值

    static void fill(type[]a, type v)

    将数组的所有数据元素值设置为 V。

    equals()是否相等

    static boolean equals(type[]a, type[]b)

    如果两个数组大小相同， 并且下标相同的元素都对应相等， 返回 true。

    17.8 多维数组

    Java 实际上没有多维数组，只有一维 数组。多维数组被解释为“ 数组的数组。 ”

    如果知道数组元素， 就可以不调用 new， 而直接使用简化的书写形式对多维数组 进行初始化

    快速地打印一个二维数组的数据元素列表

    可以调用： System.out.println(Arrays.deepToString(a));
