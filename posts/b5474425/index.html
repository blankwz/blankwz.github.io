优秀Github项目收集

不定时更新
程序都有时效性，不保证可用

综合

  • github-cheat-sheet - Github使用的各种技巧
  • GitHub-Chinese-Top-Charts GitHub中文排行榜，帮助你发现高分优秀中文项目、更高效地吸收国人的优秀经验成果
  • HelloGitHub - 分享 GitHub 上有趣、入门级的开源项目
  • pfree-programming-books - 整理了所有跟编程相关的免费书籍，而且全球多国语言版的都有
  • awesome - 整理了 GitHub 上各领域的资源大汇总
  • Free-Programming-books - [英文]各类编程书籍
  • free-programming-books-zh_CN - [中文]各类编程书籍
  • interviews - Kevin Naughton Jr.的软件工程技术面试个人指南
    • 中文版

必学

  • learn-regex -正则表达式学习 (在线练习地址)

Android

  • LearningNotes - 非常详细的面试资料，涉及 Android、Java、设计模式、算法等等等
  • android-open-project - Android 优秀开源项目集合
  • awesome-android-ui - Android UI 相关的优秀开源项目集合
  • Android_Data - 国内各种学习 Android 的资料
  • AndroidInterview-Q-A - 国内一线互联网公司内部面试题库

API

  • public-apis - 用于软件和Web开发的免费API的汇总列表

可玩

  • ohmyzsh - 最酷炫的 shell
  • huginn - 帮你实现在线自动化任务的系统
  • vscode-leetcode - 让你能在 VSCode 上刷 LeetCode
  • searchcode-server - 开源代码搜索器

  • Linux 发明者 Linus Torvalds

  • 被称为「Android之神」的 JakeWharton ， 你们用的很多开源库如 ButterKnife、OkHttp、 Retrofit、 Picasso、ViewPagerIndicator 等都是出自他之手！
