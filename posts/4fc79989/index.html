MacOS新人上手

一、快捷键

更详细请看官方说明！！

⌘ (command)
⌥ (option)
⇧ (shift)
⌃ (control)
⌫ (delete)

常用快捷键

基本与Windows差不多

  • ⌘ + Z 撤销
  • ⌘ + A 全选
  • ⌘ + S 保存
  • ⌘ + C 拷贝
  • ⌘ + V 粘贴
  • ⌘ + X 剪切 （不能剪切文件，只能剪切文本）
  • ⌘ + tab 切换应用程序

软件

  • ⌘ + Q 退出当前应用
  • ⌘ + H 隐藏当前窗口
  • ⌘ + M 最小化当前窗口
  • ⌘ + W 关闭当前窗口(并不是退出软件)
  • ⌘ + N 新建 (新建浏览器标签，等等)

不一样的快捷键

  • Space 强大的空格，预览文件！
  • ⌘ + ⌫ 将选中的文件移动到废纸篓
  • ⌘ + option + esc 强制退出某个软件。通常在软件无响应时使用
  • ⌘ + option + V 粘贴，原复制对象不会保留，相当于「移动」
  • ⌘ + Space 打开Spotlight
  • ⌘ + Control + Space 快速调用emoji表情
  • ⌘ + Control + Q 快速锁屏
  • Shift + Option + K 苹果图标

截图

  • ⇧ + ⌘ + 3 全屏幕截图
  • ⇧ + ⌘ + 4 通过鼠标选取截图

开机启动时

  • 开机长按 Option - 选择启动磁盘（双系统需要）
  • 开机长按 字母 C - 从光盘启动系统
  • 开机长按 字母 T - 火线目标盘模式
  • 开机长按 Command + Option + R + P ,一直按着直到自动重新启动两次后松开 - 重置 Mac 上的 NVRAM 或 PRAM 官方说明

也可以进入「系统偏好设置 - 键盘」，在 快捷键 下进行设置。

三、需要了解的小知识

1.OS X 不用关机
得益于 OS X 的 Power Nap 特性，在休眠期间还可以收邮件什么的。耗电也不用担心，休眠一周也基本只掉 5% 左右的电
2.强大的Spotlight
可以搜索，可以计算，可以查看汇率等等
3. 超简单卸载软件
在应用程序文件夹中，直接把 需要卸载的软件 拖拽到废纸篓即可（或者快捷键⌘ + ⌫）
4. 厉害的AirDrop ！隔空传输！（官方说明）
通过无线方式将文稿、照片、视频、网站、地图位置等发送到附近的 Mac、iPhone、iPad 或 iPod touch 上
5. Handoff 连续互通 （官方说明）
同iCloud设备间的内容续写
6. 若需Windows系统，可以装虚拟机或者双系统
7. 分屏功能
当窗口不处于全屏状态时，长按两秒左上角绿色按钮松手，即可选择右边窗口

四、技巧

技巧1：Mac与iOS设备共享屏幕

  1. 有线共享方式
    1.1 通过数据线将iPhone与Mac连接
    1.2 Mac上打开QuickTime Player
    1.3 点击菜单栏文件选项中的新建影片录制
    1.4 Mac前置摄像头会开启,点击红色按钮旁边的菜单，会出现XX的iPhone选项
    1.5 点击它，iPhone上的画面就会展现在Mac屏幕中了
  2. 无线共享方式
    2.1 Mac、iOS设备要在同一局域网下
    2.2 Mac上，下载安装并打开Airserver
    2.3 可以共享了

技巧2：MAC下的保险柜!

  1. 打开『磁盘工具』，点击上方菜单『文件』-『新建映像』-『空白映像』
    • 大小：根据自己需求写
    • 加密：128位AES加密
    • 映像格式：稀疏磁盘映像
      • 重要！这一项的含义是，镜像文件在硬盘上占的空间大小是你向镜像文件里放置的文件的实际大小，而不是上面定义的100G ，这两项结合，100G的含义是这个镜像文件所能占的最大空间
    • 格式：保持默认的 MAC OS 扩展（日志式）
    • 分区：保持默认的选项
    • 其他：随意
  2. 制作好后，双击打开，输入密码（不要记住密码）
  3. 完成！（也可以拷贝到其他苹果电脑上使用）

如果空间不够用了，可以调整大小
- 打开，磁盘工具
- 菜单栏，『映像』-『调整大小』
